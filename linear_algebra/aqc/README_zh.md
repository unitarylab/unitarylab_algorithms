# 离散绝热量子线性求解器（AQC）

## 参数设置

- `n`: 系统量子比特数（1–6），决定矩阵维度 N = 2^n。
- `T`: 离散绝热演化步数（0 表示自动选择，基于条件数 T ≈ 10·κ）。
- `p`: 绝热调度参数（> 1），默认值 1.4，控制绝热演化速率。

> **总结**：该算法使用 Trotterized 离散绝热演化在量子线路上求解线性方程组 Ax = b，通过后选择辅助量子比特寄存器为 |10000⟩ 来提取近似解。

## 运行流程

1. **问题生成**：根据给定的系统量子比特数 n，随机生成一个良态的 Hermitian 矩阵 A 和右侧向量 b，并对输入进行归一化处理。
2. **量子电路构造**：创建包含 n 个系统量子比特和 5 个辅助量子比特的电路。对每个绝热步依次施加 11 个基本操作：Hadamard 门、受控旋转、块编码、反射等。
3. **量子仿真执行**：在经典计算机上模拟量子电路的执行，得到末态。
4. **后处理与解提取**：通过对辅助寄存器后选择 |10000⟩ 态来提取系统寄存器中的近似解 x，并反归一化恢复原始问题的解。

## 核心思想

离散绝热量子线性求解器的核心策略是将线性方程组的求解过程转化为一个**绝热演化过程**。通过将目标解编码为量子基态，并从一个易于制备的初始态出发，沿着一个经过精心设计的离散化绝热路径缓慢演化，系统将保持在其瞬时基态上，最终到达编码了 Ax = b 解的量子态。

关键技术要点：
- 将矩阵 A 的求逆操作通过**块编码（block encoding）**嵌入到更大的幺正矩阵中
- 利用离散（Trotterized）绝热定理，将连续的绝热演化离散为 T 个 Trotter 步
- 每一步由 11 个子操作组成，对称地处理块编码、状态制备与反射操作
- 通过调度函数 f(s) 在演化过程中动态调整旋转角度，以平衡绝热误差与演化速度

## 数学原理

### 问题设定

求解线性方程组 $$Ax = b$$，其中：
- $$A \in \mathbb{C}^{N \times N}$$ 是 Hermitian 矩阵，$$\|A\|_2 = 1$$
- $$b \in \mathbb{C}^N$$ 是归一化的右侧向量，$$\|b\| = 1$$
- $$N = 2^n$$ 是 2 的幂次

### 绝热演化框架

构建一条参数化的哈密顿量路径 $$H(s)$$，满足：
- $$H(0)$$ 的基态是容易制备的初态（编码了向量 b）
- $$H(1)$$ 的基态包含目标解 x 的信息

通过缓慢（绝热地）变化参数 s 从 0 到 1，系统保持在 $$H(s)$$ 的瞬时基态上。

### 离散化（Trotterization）

将连续演化 $$[0, 1]$$ 离散为 T 步，每步的时间参数为 $$s = k/T$$。在第 k 步中，施加调度旋转角度：

$$\theta = 2 \arctan\left(\frac{f(s)}{1 - f(s)}\right)$$

其中调度函数定义为：

$$f(s) = \frac{\kappa}{\kappa - 1} \left(1 - \left(1 + s(\kappa^{p-1} - 1)\right)^{\frac{1}{1-p}}\right)$$

这里 $$\kappa$$ 是矩阵 A 的条件数，p 是调度参数。

### 块编码

通过 SVD 分解构造 A 的块编码幺正矩阵：

$$\begin{bmatrix} A & \sqrt{I - A^2} \\ \sqrt{I - A^2} & -A \end{bmatrix}$$

该块编码将 A 嵌入到左上角块中，为绝热演化提供对矩阵 A 的量子访问。

### 量子比特布局

采用小端序（little-endian）布局：
- 系统寄存器：量子比特 0 到 n-1（n = log₂N）
- 辅助寄存器：量子比特 n 到 n+4（5 个辅助量子比特）

### 解提取

末态通过后选择辅助寄存器处于 |10000⟩（即辅助量子比特 4 = |1⟩，其余为 |0⟩）来提取系统寄存器中的解向量。

## 算法步骤

1. **输入验证与预处理**：检查 A 的 Hermitian 性、N 是否为 2 的幂次、维度匹配等，并对 A 和 b 进行归一化。
2. **参数计算**：计算条件数 κ，若 T 未指定则自动设置 T = ceil(10·κ) 并使其为偶数。
3. **态制备**：在系统寄存器上制备归一化的 |b⟩ 态，使用 Householder 反射构造 $$U_b$$。
4. **离散绝热演化**：对 k = 1, 2, ..., T 逐次施加 11 个子操作：
   - 步骤 1：对辅助量子比特 2 施加 Hadamard 门
   - 步骤 2：CUQb1 模块（$$U_b^\dagger$$ 控制的 NOT 翻转）
   - 步骤 3：调度旋转（CZ + CRY）
   - 步骤 4：受控 Hadamard 门
   - 步骤 5：A 的受控块编码
   - 步骤 6：辅助量子比特 1 上的 X 门
   - 步骤 7：受控 Hadamard 门
   - 步骤 8：调度旋转（与步骤 3 对称）
   - 步骤 9：CUQb1 模块（与步骤 2 对称）
   - 步骤 10：辅助量子比特 2 上的 Hadamard 门
   - 步骤 11：反射操作
5. **后选择**：在辅助寄存器上后选择 |10000⟩ 态，提取系统寄存器的量子态。
6. **反归一化**：乘以内部缩放因子，并恢复原始问题的解。

## 量子优势

| 任务 | 经典方法 | 量子优势 |
|---|---|---|
| 求解线性方程组 Ax = b | 共轭梯度法 O(N κ log(1/ε)) | 对于良态稀疏矩阵，量子算法在条件数 κ 和精度 ε 方面可能实现指数级加速 |
| 矩阵求逆 | 高斯消元 O(N³) | 在特定条件下（如 κ 较小），量子算法可实现 O(poly(log N, κ)) 的复杂度 |

---

## 复杂度分析

- **量子比特数**：n_sys + 5，其中 n_sys = log₂(N)
- **门复杂度**：每个绝热步包含 11 个子操作，每个子操作的复杂度为 O(poly(n_sys))
- **总门深度**：O(T × poly(n_sys))，其中 T ≈ 10·κ
- **实用限制**：当 n_sys > 6（即 N > 64）时，多控门会使电路深度变得不切实际
- **调度参数 p**：p → 1⁺ 时绝热误差最小但演化最慢，较大的 p 值加速演化但可能增加非绝热跃迁

## 应用与影响

- **科学计算**：为大规模线性方程组的量子加速求解提供理论基础，在流体力学、结构力学等计算密集型领域具有潜在应用。
- **量子机器学习**：线性方程组求解是许多机器学习算法的核心子程序，如最小二乘回归、支持向量机等。
- **量子化学**：在量子化学模拟中，求解 Hartree-Fock 方程等线性系统是计算分子性质的关键步骤。
- **理论研究**：离散绝热方法为理解和设计更一般的量子线性代数算法（如 HHL 及其变体）提供了另一种视角和工具。
