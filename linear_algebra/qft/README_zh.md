# 量子傅里叶变换（QFT）算法详解

## 参数设置

- `n`: QFT 作用的量子比特数。
- `state`: 初始态向量，默认值为 `[1, 0, 0, ..., 0]`。若未提供，代码默认使用基态 $|0\cdots0\rangle$；若提供，则会先归一化后再加载到量子寄存器。
- `inverse`: 是否执行逆量子傅里叶变换，默认值为 `False`。当其为 `True` 时，代码会对构造出的 QFT 电路取 dagger，得到 IQFT。

> **总结**：该算法根据量子比特数 `n` 构造 QFT 或 IQFT 电路，可选择对给定初始态或默认零态执行变换。最终计算得到的结果包括量子仿真得到的末态、基于 NumPy FFT/iFFT 的参考末态、二者的验证误差以及生成的电路文件。

## 运行流程

1. **QFT 电路构造**：按照从高位到低位的顺序，对每个量子比特依次施加 Hadamard 门和多控制相位门，最后执行比特反转所需的 swap 操作；若 `inverse=True`，则对整条线路取 dagger 得到 IQFT。
2. **初态加载**：若提供 `state`，则先将其归一化后初始化到量子寄存器；否则默认使用 $|0\cdots0\rangle$。
3. **量子仿真执行**：将 QFT 或 IQFT 电路附加到寄存器上，执行量子仿真并得到最终态向量。
4. **经典 FFT 校验**：使用 NumPy 的 `ifft` 或 `fft` 计算参考结果，并与量子仿真末态做范数比较得到验证误差。
5. **结果输出**：保存最终态、参考态、误差、运行时间以及分解后的电路图与文本结果文件。

## 核心思想

QFT 是经典离散傅里叶变换在量子态空间中的幺正版本。它把计算基态的幅度重新编码到频域基中，使相位结构能够被高效提取和干涉利用。该实现通过 Hadamard 门与受控相位门逐层积累不同量子位之间的相位关系，再通过末尾的比特交换把输出顺序调整为标准 QFT 排列。

---

## 数学原理

对 $n$ 个量子比特，QFT 作用在计算基态 $|x\rangle$ 上的定义为
$$
\mathrm{QFT}|x\rangle = \frac{1}{\sqrt{2^n}} \sum_{y=0}^{2^n-1} e^{2\pi ixy/2^n}|y\rangle.
$$
其逆变换满足
$$
\mathrm{QFT}^{-1}|y\rangle = \frac{1}{\sqrt{2^n}} \sum_{x=0}^{2^n-1} e^{-2\pi ixy/2^n}|x\rangle.
$$
代码中构造 QFT 时，先对每个量子位施加 Hadamard 门，再对更低位量子位施加角度为
$$
\frac{\pi}{2^{i-j}}
$$
的多控制相位门，以累积傅里叶相位。最后交换首尾量子位，修正线路生成顺序。为了验证线路正确性，程序还会把量子仿真结果与 NumPy 的傅里叶变换结果进行比较，其中正向 QFT 对应 `ifft(state) * sqrt(2^n)`，逆 QFT 对应 `fft(state) / sqrt(2^n)`。

## 算法步骤

1. 根据量子比特数 `n` 构造标准 QFT 线路，并在需要时取其 dagger 得到 IQFT。
2. 将输入态向量归一化并加载到量子寄存器，或直接采用默认零态。
3. 执行量子傅里叶变换线路，获得量子仿真末态。
4. 使用经典 FFT 或 iFFT 计算参考结果。
5. 输出量子结果、参考结果与验证误差。

## 复杂度分析

标准 QFT 线路在 $n$ 个量子比特上通常需要 $O(n^2)$ 级别的受控相位门和交换门。本实现按显式门序逐项构造电路，因此线路深度主要由受控相位门数量决定。若采用近似 QFT，还可进一步削减小角度相位门数量，但当前实现保留的是完整精确版本，以便和 NumPy 参考结果直接比对。

## 应用与影响

- 是量子相位估计、Shor 算法和周期发现类算法的核心子程序。
- 常用于把相位信息从本征态或周期结构中提取出来。
- 也是理解量子干涉与频域编码机制的基础算法之一。
