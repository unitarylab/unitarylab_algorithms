# Amplitude Estimation (QAE) Algorithm Explained

## Parameter Settings

- `p`: Target probability, denoting the square of the probability amplitude of the "success" component in the initial quantum state, default is `0.36`.
- `d`: Denoting extraction matrix logic phases registering structures dynamically extracting numerical approximations safely resolving, default is `6`.

> **Summary**: The algorithm takes as input the probability amplitude of the 'success' component in the initial quantum state `p`, and the number of phase-register qubits `d` (default `6`, giving precision $1/2^d$). It applies the quantum amplitude estimation method: a Grover operator is constructed from `U` (a rotation unitary matrix constructed by `p`) and an Oracle, quantum phase estimation is then performed on that operator using `d` phase qubits and an IQFT to read off the phase bit-string, and the target-state amplitude is recovered from the estimated phase. The final output includes the estimated amplitude, the most likely phase bit-string together with its decimal value, the computation time, and the generated quantum circuit diagram.
