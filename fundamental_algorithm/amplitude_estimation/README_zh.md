# 量子振幅估计 (QAE) 算法详解

## 参数设置

- `p`：目标概率，表示初始量子态中“成功”分量的概率幅的平方（概率值），取值范围[0,1]， 默认值为 `0.36`
- `d`: 估算相位所依赖使用的额外量子比特大小数目（直接影响测定相位的细化分辨精度），默认值为 `6`。

> **总结**：该算法的输入包括初始态中目标态概率 `p` （默认值为`0.36`）以及相位寄存器比特数 `d`（默认值为 `6`，精度为 $1/2^d$）。算法采用量子振幅估计方法：依据概率`p`构建态制备矩阵`U`, 先由 `U` 和 Oracle 构造 Grover 算子，再以该算子为受控幺正对象执行量子相位估计，通过 IQFT 读取相位比特串，并由相位反推出目标态的振幅。最终输出估计振幅值、最可能相位比特串及对应的十进制相位、计算时间和生成的量子线路图。

## 运行流程

1. **参数制备**：设定 $d$ 个相位比特及其需要附带的用于目标判定的 Grover 辅助节点等信息提取。
2. **量子电路构建**：组装带有基底 $U$ 的完整准备步骤；构造提取靶向态概率偏差的 Grover 操作矩阵 $G$；最后并入包含该次 Grover 次数序列特征的多级受控 QPE 网络及 iQFT。
3. **量子模拟与测量**：针对整个混合系统进行底层模拟调用，并截获状态向量波形直方分布状态谱集信息。
4. **经典后处理（概率提取）**：筛选并测算最佳波峰所在的量化值 $\phi$，从而倒推计算出实际振幅和等价的事件产生理论概率限数。
5. **结果输出**：转出包含全部逻辑执行块以及估计概率信息的矢量图记录并显示日志时耗。

## 核心思想

量子振幅估计由 Brassard 等人在 2000 年提出。它巧妙结合了 Grover 迭代和量子相位估计（QPE）。由于每一个 Grover 迭代过程在本质上都包含了一个携带了成功概率角度 $2\theta$ 的固有量子反向偏转旋转因子，QAE 通过对整个旋转系统应用 QPE 策略读取该频率序列的相位大小，从而直接倒推获得事件概率的极其精准的估计值，而不用耗费成千上万次单独采样。

## 数学原理

- **设定**：令 $U\lvert 0 \rangle = \sqrt{a}\,\lvert\text{good}\rangle + \sqrt{1-a}\,\lvert\text{bad}\rangle$，其中 $a\in(0,1)$ 即待估计的成功概率。
- **角度参数化**：写出分解 $\sqrt{a}=\sin\theta,\ \sqrt{1-a}=\cos\theta$，其中 $a=\sin^2\theta$ 使得 $\theta\in(0,\pi/2)$。
- **反射与Grover 迭代**：好态反射 $S_f$ 与 $\lvert\psi\rangle$ 反射生成 $Q = S_{\psi} S_f$。在二维子空间中，$Q$ 拥有包含角度信号的本征基系：
  $$|\psi_\pm\rangle=\frac{1}{\sqrt2}\big(|\text{good}\rangle\pm i|\text{bad}\rangle\big),\quad Q|\psi_\pm\rangle=e^{\pm i2\theta}|\psi_\pm\rangle.$$
  其本征值对应的纯内生相位 $\pm 2\theta$ 完整携带了所有的目标偏转发生信息及等效发生几率。
- **转化为相位角**：对算符单元 $Q$ 完成一整套 QPE 操作，进而测得频率参数 $\phi \approx 2\theta / (2\pi)$（取正号本征值极限提取）。
  可推算恢复：$$\theta \approx \pi\phi,\qquad \hat a = \sin^2(\pi\phi).$$
- **精度误差**：相位寄存器位数 $d$ 给出分辨率界点 $\delta\phi \approx 1/2^d$，对应幅度误差量级最坏保障处在 $\delta a \approx \pi\, \delta\phi$。主峰捕获依然维持约 $P_{\text{peak}} \approx 0.405$ 的理论有效绝对最高信噪。

## 算法步骤

1. 在量子运算主进程中引入初始化生成域，建立由 $U$ 控制的完整概率波。
2. 串联布置一个受控的并行评估阶段（QPE）使得各个探测层依照二进制 $2^k$ 倍率嵌套式叠加 Grover 搜索块。
3. 对生成的全控制网络尾端采用标准的 iQFT（逆量子傅里叶转换）进行最终状态干涉坍缩转换。
4. 从高置信最高频探测峰处反推其小数位频率并得出其初始等效 $\sin^2$ 波形原始发生概率。

## 复杂度分析

查询复杂度：一次 QPE 中相位寄存器第 $k$ 位控制施加需要调用门电路量级等效于 $Q^{2^k}$，总体需要调用 $O(2^d)$ 次 Grover 迭代；严格等价于总体 $O(1/\epsilon)$ 次调用 $Q$ 即可把误差缩减到 $\epsilon$。而最先进的高级经典蒙特卡洛估计概率通常最低都必须耗散 $O(1/\epsilon^2)$ 次海量冗杂独立采样规模。

## 应用与影响

它是极为重要的蒙特卡洛积分求解的核心高阶提速子架构层工具！主要解决了传统分析体系在需要极致精确度测定微量随机发生偏向事件几率时引发的指数爆炸负担问题，为复杂科学定价分析和数值估计等前沿衍生课题扫清了主要性能约束障碍。
