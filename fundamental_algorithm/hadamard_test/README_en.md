# Hadamard Test Algorithm Explained

## Parameter Settings

- `U`: Designated primary unitary base, default `[[1, 0], [0, -1]]`. 
- `Psi`: Preparing psi state components, default `[1, 2]`. 
- `shots`: Evaluating measurement shots mathematically, default `0`, for exact expectation value calculation.

> **Summary**: The algorithm takes as input the unitary operator `U`, state-preparation circuits `Psi`, and the number of measurement shots `shots` (default `0`). It applies the Hadamard test method: a test circuit with one ancilla qubit and a controlled unitary is built according to the selected mode, and the ancilla measurement outcomes are combined with statistical sampling to estimate the expectation value, state overlap, or phase. The final output includes the estimated real or imaginary part of the expectation value, the state-overlap fidelity or complex phase (depending on mode), the computation time, and the generated quantum circuit diagram.
