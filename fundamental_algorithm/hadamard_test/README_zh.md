# Hadamard 测试 (Hadamard Test) 详解

## 参数设置

- `U`: 直接控制用于验证的主体发生门阵部件，默认值为 `[[1, 0], [0, -1]]`。
- `Psi`: 进行基准判定目标原始参照态制备电路，默认值为 `[1, 2]`。
- `shots`: 量子取样测算运行总次数，默认值为 `0`，表示进行精确计算。

> **总结**：该算法运行模式为求期望，输入默认幺正算符 `U`、态制备电路 `Psi` 以及统计采样数 `shots`（默认值为 `0`）。算法采用 Hadamard 测试方法：根据所选模式构造辅助比特加控制幺正的测试线路，通过辅助比特测量结合统计采样计算锚定期望值、态交叠或相位。最终输出期望值的实部或虚部估计、态交叠保真度或复数相位，以及计算时间和生成的量子线路图。

## 运行流程

1. **参数制备**：系统审核检验指定提取方式模式的完整性（包括是否有附带验证目标块存在等），设置概率采集所需的重复测试次数标的设置。
2. **量子电路构建**：绑定挂载测试目标网路以及参照组特征模块的独立分布网络；如果是包含提取虚数的命令分支设定，则在其中专门挂载引入包含 $-i$ 极化的推斥阵干涉节点并链接对应的受控制核心。
3. **量子模拟与测量**：发射全局并获取位于根节点的最前端附属辅助探测比特位点的输出倒相情况发生率及对应次数频率分布谱。
4. **经典后处理（估算提取）**：根据不同指定的 `mode` 选择运用适当对应公式，执行简单映射通过统计散粒噪声及正相关比例转化算出原始实数/虚数的极点数值分布比例甚至得到完整的两方交叠保真面积比率平方数值。
5. **结果输出**：综合打出生成逻辑运算网路线结构与精确测量测算波段总结日志展示图表参数结果报告供审验。

## 核心思想

Hadamard 测试是一种基础的量子干涉辅助统计算法，设计极尽精简但十分强悍有力并且可以非常高效且精确地用于统计及测量一个酉算子 $U$ 在某个特定的基准加载体系量子态 $|\psi\rangle$ 测试环境下的各种复数量及本征值属性表现！这种测验并不是去粗暴测量那个本体寄存网络内的态如何发生坍缩进而干扰并毁灭该波段态的本身，而是只关注附着其旁的辅路的相干变动。这个电路的核心设计"神奇点"就是：由于其通过运用了附带连接的全局量子“受控-$U$”逻辑体系进行驱动操作，使得本体干涉控制会导致把 $U$ 自身的作用内卷变成了对于源头控制极 ancilla 位状态分岔情况方向发生的相对独立特征推相进而拉产生了相对幅度频率的差距，从而在这个专门设计的小型探测点极位测量数据频率差上直观清晰完美体现出来被测网络的隐秘属性数据。

## 数学原理

对于一般期望提取（给定计算期待 $\langle\psi|U|\psi\rangle$ 以及被试酉序列阵）：
- 实部提取模式：通过测定最终 ancilla 发射探测节点得到最终呈现值为 0 或者 1 发生状态频次极限率结论：
  $\langle Z\rangle_{\text{anc}} = p(0)-p(1)=\mathrm{Re}\left(\langle\psi|U|\psi\rangle\right)$
  其中 $\langle Z\rangle_{\text{anc}}$ 的意思代表着在此控制判定点辅助态做了单纯的基础状态极测计算得到的预期偏平偏量。
- 虚部提取模式（imag Hadamard test）：需要计算纯虚数占比度影响的时候，需要强制改变控制比特极位并在第一道启动探测 $H$ 并紧随附加单点的负极极相干涉节点如插入作用带有一个附加 $1/2$ 相向门参数或者加上直接偏转极算符如加上 $S^\dagger$，最终获得偏差结果呈现的是绝对的截然不同的：
  $\langle Z\rangle_{\text{anc}} = \mathrm{Im}\left(\langle\psi|U|\psi\rangle\right)$
  依靠分别采集提取并进行叠加就能在外部极其简单快速恢复整个复合叠加系统黑箱域内存在的任何完整的复数体系波段及全部细节重组（$\text{cos\_est} + i\,\text{sin\_est}$）。

而在涉及到进行非常经典的两组体系或态结构相似评估交叠分析时（即运用 Swap-Test 的干涉应用版衍生）：
此方案在判定测量端通过同样的探测最终生成的概率密度直出得到的目标差则精确刻画了目标被测双方系统完全一致互相印证程度面积比例指标 $\langle Z \rangle = |\langle\phi|\psi\rangle|^2$ 。

## 算法步骤

1. 设置用于干涉检测比较的核心单独量子工作指示测试锚定位并将其激活出完整的叠加干涉初始发射特征面（部署极位 H）。
2. 布置待测试评估模块及其比照对比标本（如涉及）部署装满关联量子特征阵内环境体系态参数特征进入准备面加载区。
3. 从探测锚点接出控制干线并在测算极上链接被受激触发并激活网络。
4. 给标的点进行还原碰撞坍塌退相干并做出判定测量统计收集出最终波峰偏移和频段信息完成评估。

## 复杂度分析

这是一种堪称极其令人震撼的超级“高维计算探针微创手术”网络控制构架解析方式算法！这种测算法具有极低且可以忽略不计的空间计算成本拓展需求（**永远都只仅仅多借用和占用了微不足道的唯一 1 个核心额外单点辅助判定量子引线位！**）这种极其微小的测试架构和探头就足够测量甚至可以极其恐怖无限制无上界容纳高维结构运算体的深层空间体系参数而完全不在乎整体状态系统的复杂度深厚度和海量交织！其限制精度和精确时间收敛能力仅仅只受最终人为采样要求数控制并在极大范围 $O(1/\epsilon^2)$ 上可以依靠大量直接且廉价的物理发生取样快速实现均值逼近。
