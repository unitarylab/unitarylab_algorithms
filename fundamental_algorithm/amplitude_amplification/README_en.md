# Amplitude Amplification Algorithm Explained

## Parameter Settings

- `p`: Defined initial success probabilities used deriving cyclic scaling steps recursively.
- `reps`: Optional manual assignment bypassing autonomous calculations providing fixed iterations instead, default is `None`.

> **Summary**: The algorithm takes as input the initial state preparation circuit `U`, the zero-qubit index set `good_zero_qubits` that defines the target state, the initial success probability `p`, and an optional iteration count `reps`. It applies the amplitude amplification method: `U` first prepares the initial quantum state, an Oracle then marks basis states satisfying the target condition, and a diffuser repeatedly performs Grover iterations to boost the measurement probability of the target state. The final output includes the amplified target-state probability, the actual number of iterations used, the computation time, and the generated quantum circuit diagram, which together verify whether the target state has been successfully amplified relative to the initial probability.
