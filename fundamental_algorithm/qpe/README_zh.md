# 量子相位估计 (QPE) 算法详解

## 参数设置

- `phase`: 输入的相位值，默认 `0.25`。
- `n`: 用于执行提取辅助精度的控制寄存器分配规模量即比特数。

> **总结**：该算法的输入为相位值 `phase`、相位寄存器比特数 `n`（精度为 $1/2^n$）。算法采用量子相位估计方法：相位寄存器先经 Hadamard 制备均匀叠加，再通过一系列受控 $U^{2^k}$ 门将相位信息写入相位比特，最后由 IQFT 读取相位比特串并换算为十进制估计相位。最终输出估计相位值、最可能相位比特串及其测量概率、计算时间和生成的量子线路图。

## 运行流程

1. **参数制备**：构建必要的控制模块宽度与配置加载被测幺正主环境，测算理论分辨角频。
2. **量子电路构建**：从上至下搭建测试结构寄存器并触发全频域哈达玛展开；关联构建一系列经过精巧控制位触发不同量级的受控幺正乘方循环矩阵网络；并闭合适配专属的 `iQFT` 输出重组。
3. **量子模拟与测量**：针对整个组装后的框架发起测试进程，生成基态演进时耗及其对应向量组别结果反馈表。
4. **经典后处理（频段解析）**：截获相位域统计信息流后选取包含最高出现频次的观测本征基；转化比对求解具体频段小数输出并计算偏转估计。
5. **结果输出**：系统打印相关测定波段、误差与最终测算参数，打包最终绘制好的 SVG 测试模型图。

## 核心思想

考虑有特定的酉矩阵算符 $U$ 以及由于 $U$ 操作带来的关联旋转角响应，我们的目的是找出精准度控制良好的近似解 $\phi$。作为多重架构底层工具，QPE 主要原理是将酉演化转换受控模式执行，这能在整个辅助工作控制比特序列上留下特定的累加旋转响应因子。而在提取结果时充分借用离散的逆向傅里叶（IQFT）强大的频率反转重构能力将其从混乱波干涉形态完整翻译回十进制可视位分布从而解算出这个具体的物理数值。

## 数学原理

对于第 $j$ 个计算基底 $\ket{j}$，我们定义离散的量子傅立叶变换为
$$U_{FT}\ket{j} = \frac{1}{\sqrt{N}}\sum_{k\in[N]}e^{i2\pi\frac{kj}{N}}\ket{k}$$
在问题推导中假定 $\ket{\psi}$ 是 $U$ 的特征向量，也就是：
$$U\ket{\psi} = e^{i2\pi\phi}\ket{\psi},\ \ \phi\in[0,1)$$
考虑如下的多控量子门结构分解作用规律：
$$\mathcal{U} = \prod_{i=0}^{d-1} \left( \lvert 0\rangle\langle 0\rvert \otimes I_n \;+\; \lvert 1\rangle\langle 1\rvert \otimes U^{2^i} \right)$$
受此影响结合后的态将包含有丰富的干涉推导过程与结果形态记录如下面公式演算推导：
$$\lvert 0^d\rangle \lvert \psi_0\rangle \xrightarrow{\,U_{\mathrm{FT}}\otimes I\,} \frac{1}{\sqrt{2^d}}\sum_{j\in[2^d]} \lvert j\rangle \lvert \psi_0\rangle \xrightarrow{\,\mathcal{U}\,} \frac{1}{\sqrt{2^d}}\sum_{j\in[2^d]} \lvert j\rangle\, e^{i2\pi \varphi j}\lvert \psi_0\rangle \xrightarrow{\,U_{\mathrm{FT}}^{\dagger}\otimes I\,} \lvert k'_{0}\rangle \lvert \psi_0\rangle$$
对于求取的相位 $\phi$ 计算后有极其高的概率落在 $\vert \phi_{approx}-\phi\vert\leq2^{-d}$ 置信近似值区间，因此该估计几乎是严格的绝对收敛提取方法。

## 算法步骤

1. 对所有分配在附加层上执行用于测量相位的空位初始化并行化，令所有的控制项包含最大的重合概率分布区位。
2. 配置不同的序列连乘循环数 $U^{2^k}$，执行由浅入深不同深度的递增特征偏转。
3. 执行 $iQFT$ 算符操作从而从频域重新回到实数值空间。
4. 对上层辅助控制位全部投影测量提取近似态波段。

## 复杂度分析

复杂度判定：如果在复杂向量分析体系中对于传统计算机去实现快速傅立叶变换和频响分析必须要长达 $O(N\log\ N)$ 次循环。如果在量子内构建仅需耗费 $O(\log^{2}\ N)$ 复杂度！
如果要求测定误差极小处于 $\epsilon$ 量级底限区间，那么需求满足 $d\geq \log\ \frac{1}{\epsilon}$，其总体操作深度大致处在优良的 $O(\frac{1}{\epsilon})$ 幅度水平上。

## 应用与影响

它是诸如寻找量子因数核心周期算符的根源推动器，包含著名的 Shor 算法及解决许多偏分大型特异方程甚至大型化学网络基础结构模型测算求解，都在利用 QPE 作为其不可获取甚至居于心脏地位的中流砥柱。
