# CVQNN 算法

## 参数设置

- `layers`：变分层数，默认值为 `2`。
- `cutoff`：Fock 空间截断维度，默认值为 `6`。
- `epochs`：训练迭代次数，默认值为 `40`。
- `lr`：Adam 优化器学习率，默认值为 `0.05`。

> **总结**：该算法使用连续变量量子神经网络对二维训练数据进行二分类。它先对输入特征做标准化，再在截断 Fock 空间中构造位移、压缩、旋转、Kerr 非线性和类分束器耦合组成的 CV 模型，并用 Adam 优化模型参数。最终计算得到的结果包括最终训练损失、最终分类准确率、总计算时间，以及生成的量子电路图和训练指标图。

## 运行流程

1. **数据预处理与模型初始化**：对输入特征做标准化，把标签转换为训练目标，并初始化 CVClassifier、Adam 优化器和均方误差损失函数。
2. **量子相空间结构构建**：根据 `layers` 生成一个用于展示拓扑的量子电路，并在模型内部准备连续变量量子层参数。
3. **变分训练循环**：在每个 epoch 中执行前向传播、损失计算、反向传播和参数更新，并记录损失历史。
4. **分类评估**：用训练完成后的模型重新预测样本类别，并计算最终准确率。
5. **结果导出**：保存量子线路图、训练损失与决策边界图，并输出最终指标。

## 核心思想

CVQNN 的核心思想，是把经典连续特征编码为连续变量量子态上的位移操作，再通过多层可训练的高斯与非高斯量子操作变换这些态，最后从某个可观测量的期望值中得到分类输出。和基于量子比特的模型不同，这里直接在截断的 Fock 空间里模拟光学连续变量模式，因此更贴近连续信号处理的建模方式。

## 数学原理

连续变量量子神经网络（CVQNN）工作在量子谐振子的 Fock 空间表示上。代码中的 `CVSimulator` 首先构造湮灭算符 `a`、产生算符 `adag`、位置算符 `x_op` 和数算符 `n_op`，并将真空态作为初始参考态。每个输入样本的两个特征被编码为两个模式上的位移操作，从而得到两模输入态。

在每一层变分结构中，模型对两个模式分别施加旋转、压缩、位移和 Kerr 非线性操作，并通过一个类似分束器的双模耦合变换连接两个模式。经过 `layers` 层迭代后，代码计算第一个模式位置算符的期望值
$$
\langle x \rangle
$$
并把它作为分类输出。训练阶段使用均方误差损失，将标签从 `{0,1}` 映射到 `{-1,1}`，再用 Adam 对连续变量量子层参数进行优化。

---

## 算法步骤

1. 对训练特征做标准化，并把标签映射到训练目标值。
2. 用位移操作把每个二维样本编码为两模连续变量量子态。
3. 通过多层可训练的单模门和双模耦合门演化量子态。
4. 以位置算符期望值作为输出，并用 MSE 损失训练参数。
5. 统计最终准确率并导出损失曲线与决策边界图。

## 复杂度分析

该实现的计算成本主要由三部分决定：Fock 截断维度 `cutoff`、变分层数 `layers` 和训练轮数 `epochs`。截断维度越大，每个模式对应的矩阵维度越高；层数越多，前向传播中的矩阵指数与双模 Kronecker 结构就越复杂；训练轮数越多，总反向传播次数也越多。因此，该实现更适合作为 CVQNN 训练流程与相空间建模方法的演示原型。

## 应用与影响

- 适合演示连续变量量子机器学习在二维连续特征分类上的工作流程。
- 体现了位移、压缩、Kerr 非线性和模间耦合在 CV 模型中的组合方式。
- 可作为进一步研究光学量子神经网络和连续变量变分学习的基础实现。
