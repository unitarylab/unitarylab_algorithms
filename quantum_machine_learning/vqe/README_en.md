# Variational Quantum Eigensolver (VQE)

## Parameter Settings

- `n`: Number of qubits. The default is `2`. The result-analysis stage currently exports the `Z0`, `Z1`, and `Z0Z1` observables for the 2-qubit case, so the default example is also used as a 2-qubit transverse-field Ising model.
- `layers`: Number of variational layers. The default is `3`.
- `max_iter`: Maximum COBYLA iterations. (default is `150`).

> **Summary**: This algorithm uses the Variational Quantum Eigensolver to estimate the ground-state energy of a transverse-field Ising Hamiltonian. The code first constructs the Hamiltonian and a parameterized circuit made of `Ry` rotations and `CX` entangling gates, then calls COBYLA to minimize the energy expectation value, and finally evaluates several spin observables after optimization. The final calculated result includes the exact ground-state energy, the optimized energy, the expectation values of `Z0`, `Z1`, and `Z0Z1`, the core quantum-computation time, and the generated circuit, convergence, and observable plots.

## Execution Flow

1. **Hamiltonian and Initial-Value Preparation**: Build the transverse-field Ising Hamiltonian, compute the exact ground-state energy, and randomly initialize the variational parameters.
2. **Variational Circuit Construction**: Generate an ansatz composed of repeated `Ry` rotations and `CX(0,1)` entangling gates, and save the circuit diagram.
3. **Classical Optimization Loop**: Use the quantum-state energy expectation value as the objective function and minimize it with COBYLA.
4. **Post-Optimization Observable Analysis**: Re-run the circuit with the optimized parameters and evaluate the `Z0`, `Z1`, and `Z0Z1` expectation values.
5. **Result Export**: Save the energy-convergence plot, the observable bar chart, and the circuit diagram.

## Core Idea

The core idea of VQE is to recast a hard ground-state problem into a hybrid optimization loop: a quantum circuit prepares a trial state, and a classical optimizer adjusts the circuit parameters to minimize the measured energy. The quantum part represents candidate wavefunctions and evaluates expectation values, while the classical part updates the parameters so that the trial energy approaches the true ground-state energy.

---

## Mathematical Principles
Variational Quantum Eigensolver (VQE) is a hybrid quantum-classical method for approximating low-energy eigenstates of a Hamiltonian, especially the ground state. In the current implementation, the target Hamiltonian is the transverse-field Ising model
$$
H = \sum_{i=0}^{n-2} Z_i Z_{i+1} - 0.5 \sum_{i=0}^{n-1} X_i,
$$
and the trial state $|\psi(\theta)\rangle$ is generated by a parameterized quantum circuit.

By the variational principle, the energy expectation value of any trial state is an upper bound on the exact ground-state energy:
$$
E_0 \le \langle \psi(\theta) | H | \psi(\theta) \rangle.
$$
Therefore, minimizing this expectation value with respect to the parameters $\theta$ drives the trial state toward the ground state. In this implementation, each ansatz layer applies one `Ry` rotation to every qubit followed by a fixed `CX(0,1)` entangling gate, and the optimizer is COBYLA, so no explicit gradient is required.

## Algorithm Steps

1. Construct the transverse-field Ising Hamiltonian and obtain the exact ground-state energy by eigendecomposition for reference.
2. Initialize the variational parameters and build the `Ry` + `CX` ansatz circuit.
3. Execute the circuit in each iteration and compute the energy expectation value of the output state.
4. Update the parameters with COBYLA until the iteration budget is reached or convergence is achieved.
5. Evaluate the final energy and spin observables with the optimized parameters, then export the plots.

| Task | Classical Method | VQE Advantage |
|---|---|---|
| Ground-state energy estimation | Exact diagonalization becomes very expensive for larger quantum many-body systems | Uses a shallow parameterized quantum circuit to approximate the ground state and estimates the energy directly from quantum measurements |

## Complexity Analysis

The circuit depth is mainly controlled by `layers`, while each objective-function evaluation requires a full state evolution and an energy-expectation calculation. The total runtime is therefore largely determined by the number of COBYLA objective evaluations up to `max_iter`, together with the simulator cost of handling a `2^n`-dimensional state vector and Hamiltonian matrix. Since this implementation uses classical simulation, both memory and runtime grow exponentially with `n`; the default 2-qubit configuration is the practical demonstration setting.

## Applications and Impact

- It is useful for demonstrating how hybrid quantum-classical optimization approximates ground-state energies.
- It serves as an introductory example for the Ising model, VQE ansatz design, and COBYLA-based optimization.
- It can be extended to more complex Hamiltonians, richer ansatz circuits, and broader observable analysis.
